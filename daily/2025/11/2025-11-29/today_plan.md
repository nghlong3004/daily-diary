## Deploy
### Hiểu về "môi trường sống" của ứng dụng
- Cách connect tới 1 server ?
- Câu lệnh trong Ubuntu ra sao?
- Config những gì ?
- Cách chạy 1 project Spring
- Cách follow log của project đấy
- monitoring ra sao ?
### Deploy thủ công
- chuẩn bị môi trường
- chuẩn bị code
- chạy ứng dụng
    - Vấn đề: khi tắt terminal, web cũng tắt theo
    - giải pháp: học dùng process Manager
- Web server (reverse proxy)
    - học Nginx, Tomcat
### Domain & Security (HTTPS)
- Tìm hiểu về domain và cáh trỏ DNS về IP server
- SSL/TLS: chứng chỉ bảo mật
    - Công cụ: Certbot
### Dóng gói
- Tìm hiểu về Docker:
    - Docker file
    - docker build, docker run
    - Docker Compose: để chạy cùng lúc cả DB và App
### Tự động hoá (CI/CD)
- Vấn đề: mỗi lần sửa code lại phải SSH vào server mất thời gian
- CI (continuous Integration): Tự động test khi đẩy lên github
- CD (Continuous Deployment): Tự động đảy code mới vào server và restart lại app
